Introducción al HTML

¿Qué es HTML y para qué sirve?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear y estructurar el contenido de las páginas web. Permite definir la estructura y el contenido mediante etiquetas y elementos que indican cómo debe mostrarse y organizarse el contenido, como encabezados, párrafos, enlaces, imágenes y más.

Un poco de Historia

El origen de HTML se remonta al año 1980, cuando el físico Tim Berners-Lee propuso un nuevo sistema de "Hipertexto" para compartir documentos. Este sistema se convirtió en el precursor de las páginas web actuales. Tras unir fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb (W3). El primer documento formal con la descripción de HTML se publicó en 1991 bajo el nombre de "HTML Tags" (Etiquetas HTML). En 1995, se publicó el estándar HTML 2.0 por parte del organismo IETF (Internet Engineering Task Force).
